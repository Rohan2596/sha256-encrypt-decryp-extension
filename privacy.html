Privacy Policy

Last updated: 2025

Overview

SHA-256 Encrypt / Compare is a browser-based Chrome extension designed to generate and compare SHA-256 hashes entirely on your local device.

Developer Information

Developer:Rohan Ravindra Kadam
Contact: rohankadam2596@gmail.com

Data Collection

This extension does not collect, store, transmit, or share any personal or sensitive user data.

Data Processing

All text entered into the extension is processed locally using the Web Crypto API provided by your browser. No data is sent to external servers.

Permissions

This extension does not require any special permissions.

Third-Party Services

This extension does not use any third-party analytics, trackers, advertising networks, or external APIs.

Offline Use

The extension works completely offline once installed.

Any future updates to this privacy policy will be reflected on this page.
